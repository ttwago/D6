Exercise:

In the course of working with the module, you had to allow the task to run:

Create a new page at /news/ that should list all the news.
Output all articles as title, publication date and first 20 characters of text.
News should be displayed in order from newest to oldest.
Make a separate page for full information about the article /news/<news id>.
This page should contain all the information about the article. Title, text and date of upload in day.month.year format.

Write your own censor filter that replaces letters with unusual words in the titles and texts of articles with the symbol "*".

All new pages should use the default.html template as a base.
